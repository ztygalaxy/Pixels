## Pixels

#### 🌏 与世界分享你的英雄梦

一个基于 Android 手机端的漫威元素图片交流社区APP `Pixels`。

#### 🔨 主要功能

- 版本控制
- Json 数据传输
- 图片上传与下载
- 基于 [Makers](https://github.com/dsandler/markers) 的多功能画板
- 意见反馈
- LRUCache 图片缓存

#### 📕 目录结构

- 移动端 Android
- 基于 Servlet MVC 的服务端 Server
- 基于SSM框架重构的服务端 Pixels_server 

#### 🚆 开发与测试平台

- Android Studio
- Eclipse
- IDEA
- Genymotion
